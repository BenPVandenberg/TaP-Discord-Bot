# T&P-Discord-Bot

## Structure

| Codebase |     Description |
| :------- | --------------: |
| Envy     |  React frontend |
| gala     | express backend |
| opal     |           mySQL |

## How to run locally

1. Create a .env file following the example (if gala)
2. Run ```npm install``` in envy or gala
3. Run ```npm start```
